# FasterEEG
This repository hold the codebase for our TBME 2025 paper:  FasterEEG: Adaptive Channel and Model Size Optimization for Efficient Brain Decoding, Jiaxing Wang, Weiqun Wang, Chutian Zhang, and Zeng-Guang Hou.
![Framework](https://github.com/user-attachments/assets/7fbc6eb7-3459-46d2-89e6-b60205432cdd)
# Dataset
All experiments are conducted on publicly available EEG datasets, including BCI competition IV 2a dataset (BCI IV 2a：https://www.bbci.de/competition/iv/) and High Gamma Dataset (HGD：https://github.com/robintibor/high-gamma-dataset?tab=readme-ov-file).
